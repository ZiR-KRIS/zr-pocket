# ZR Pocket

Cascarón público de la PWA "ZR Pocket" — capa de vista y captura móvil sobre el repo privado `zr-code`.

Este repo solo contiene código (HTML/CSS/JS vanilla, sin build step). Los datos reales de ZiR viven en `zr-code` (privado) y se leen/escriben vía GitHub API con un token personal guardado solo en el dispositivo — nunca en este código.
